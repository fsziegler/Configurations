## Git Commands

### git add [options ...] [<pathspec> ... ]

#### Options

##### Informational
* [--verbose | -v]
* [--dry-run | -n]

##### User interaction
* [--interactive | -i]
* [--patch | -p]
* [--edit | -e] 


* [--force | -f]
* [--[no-]all | --[no-]ignore-removal | [--update | -u]]	  
* [--intent-to-add | -N] 
* [--refresh] 
* [--ignore-errors] 
* [--ignore-missing]
